### BigImageCheck-SDWebImage

检测SDWebImage下载的大图，图片大小超过预定义大小，会在图片上显示“大图 + 图片尺寸”提醒文字，只在Debug下生效，方便开发期间监控大图。


### 接入

pod 'BigImageCheck-SDWebImage', '~> 0.1.3'
